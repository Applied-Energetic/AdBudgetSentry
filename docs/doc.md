# 快手磁力金牛助手脚本解析文档

本文档旨在解析“快手磁力金牛助手-2.1.user.js”油猴脚本的功能、适用范围、使用方法以及定制指南。

## 1. 脚本功能介绍

该脚本是一个辅助运营工具，主要用于**快手磁力金牛**广告投放平台的管理后台。它通过在页面上注入一个悬浮控制面板，允许用户设置一系列自动化规则，脚本会定时监控页面上的广告数据（如花费、ROI、转化成本等），并根据设定的规则自动开启/关闭广告计划或调整预算。

### 主要功能点：

1.  **秘钥保护**：首次使用需要输入特定秘钥（默认：`aMgXfHF.4k.VBH4`）才能激活功能。
2.  **自动分页优化**：自动将页面的表格分页大小设置为 200 条，方便一次性监控更多数据。
3.  **悬浮控制面板**：提供一个可拖拽的 UI 面板，用于参数配置、功能开关和手动执行。
4.  **数据实时监控**：
    *   自动识别当前页面类型（专业版/经营版，计划/单元/创意层级）。
    *   定时抓取表格关键数据：**花费、直接ROI、转化成本、当日累计订单数、今日预算**。
5.  **自动化策略（5种核心策略）**：
    *   **策略1（成本控制）**：设定平均花费阈值，若新增花费超过阈值且未出单，则自动关闭；满足条件则开启。
    *   **策略2（低 ROI 止损）**：当日 ROI 低于设定值时自动关闭广告，否则开启。
    *   **策略3（高转化成本止损）**：当日转化成本高于设定值时自动关闭广告，否则开启。
    *   **策略4（无单止损）**：当日花费超过设定金额但累计订单数为 0 时，自动关闭广告。
    *   **策略5（自动扩量）**：当“经营版/专业版广告组”层级的当日花费超过预算的一半时，自动将预算扩大指定倍数。

## 2. 脚本适用范围

*   **平台**：快手磁力金牛投放平台
*   **网址匹配**：`https://niu.e.kuaishou.com/manage*`
*   **支持的页面层级**：
    *   **经营版**：广告计划、广告创意
    *   **专业版**：广告计划、广告组、广告创意
*   **浏览器环境**：支持 Tampermonkey（油猴）扩展的现代浏览器（Chrome, Edge 等）。

## 3. 脚本使用指南

### 3.1 安装与激活
1.  在浏览器安装 Tampermonkey 插件。
2.  安装本脚本。
3.  打开快手磁力金牛管理后台，刷新页面。
4.  首次运行会弹出遮罩层提示输入秘钥，输入 `aMgXfHF.4k.VBH4` 并点击确定。

### 3.2 界面操作
页面加载完毕后，左下角会出现名为“**当前监听页面**”的悬浮窗：

*   **刷新间隔**：设置脚本扫描数据和执行操作的时间间隔（默认为 10 秒）。
*   **功能勾选**：
    1.  勾选需要启用的策略（1-5）。
    2.  在对应的输入框中填入阈值参数（如 ROI 阈值 0.8，花费阈值 100 等）。
*   **按钮**：
    *   **保存并执行**：保存当前配置到浏览器缓存，并立即开始按照规则执行自动化操作。
    *   **清空当前页面缓存**：清除当前页面层级的历史数据记录（用于重置统计基准）。

### 3.3 运行逻辑
*   脚本会首先按“花费”列进行降序排列，确保重点关注高花费计划。
*   脚本会根据配置的间隔时间，循环检查表格中的每一行数据。
*   **注意**：脚本会自动模拟点击操作（点击开关或修改预算），请在无人值守运行时保持页面前台运行。

## 4. 如何根据自己的需求定制脚本

如果您懂一点 JavaScript，可以通过编辑脚本代码来满足个性化需求。

### 4.1 修改秘钥
如果不希望被秘钥验证打扰，或者想修改秘钥：
*   **定位代码**：搜索 `checkMM` 函数或 `secretKey == 'aMgXfHF.4k.VBH4'`。
*   **修改方法**：直接将判断条件改为 `true`，或修改字符串内容。

### 4.2 调整默认刷新间隔
*   **定位代码**：在 `createGUI` 函数中找到 `$('#i0').val('10');`。
*   **修改方法**：将 `'10'` 改为您想要的默认秒数。

### 4.3 修改自动化策略逻辑
核心逻辑位于 `doMain` 函数中。
*   **场景示例**：比如您想修改“策略2（ROI控制）”，使其仅在花费大于 50 元时才生效。
*   **定位代码**：
    ```javascript
    if (chk_value.indexOf('2') >= 0 && i2 != null && i2 != '') {
       // 在这里增加花费判断逻辑
       if (parseFloat(item.cost) > 50 && parseFloat(item.roi) < parseFloat(i2) && parseFloat(item.roi) > 0) {
           close = 1;
       } else {
           open = 1;
       }
    }
    ```

### 4.4 增加新的监控字段
如果需要监控除了 ROI 和花费之外的指标（例如“点击率”）：
1.  **定位表头解析**：在 `initInterval` 函数中，找到遍历 `headerDom` 的循环，增加 `点击率` 的列索引记录。
2.  **定位数据读取**：在 `initInterval` 的数据抓取循环中，增加 `ctr: $(css_row + '>td:nth-child(' + ctr_i + ')').text()`。
3.  **应用策略**：在 `doMain` 函数中增加新的判断逻辑使用该字段。

### 4.5 调整 UI 样式
*   **定位代码**：`createGUI` 函数中的 `html.push` 部分。
*   **修改方法**：可以直接修改 HTML 字符串中的 `style` 属性来调整悬浮窗的大小、颜色或位置。
